# ACTIVITY INSTRUCTIONS 

The Case of "Justice & Associates"
The Story: Lead attorney Elias Thorne is tired of his firm looking "stuck in the 90s." 
He wants a site that feels as heavy and serious as a mahogany desk.

The Mission: Create a 4-page site.
  
  Page 1 (The Verdict): A high-contrast landing page with  Hero section with a high-contrast background image. 
  Use <h1> for the firm name and a <section> for "Why Choose Us."
  
  Page 2 (The Docket): Use an HTML <table> to list case types and hourly rates, services (Criminal, Civil, Corporate) 
  and their primary consultants. Use padding and border-bottom for row separation.
  
  Page 3 (The Partners): Use <article> and flexbox to create "Lawyer Cards." Use <article> tags for each lawyer. 
  Include a <figure> for their portrait and a <ul> for their credentials.
  
  Page 4 (The Deposition): A contact form using <fieldset> to separate personal info from case details.
  A <form> with fieldset and legend grouping "Personal Info" and "Case Details." Integrate an <iframe> for a Google Maps 
  location that each client can use in contacting the law firm
